### Code Convention

Publish new version, automatically lint and build, for Mac OS: 
>$ npm run pb

for windows:
>$ npm run pb-cmd

Only lint 
>$ npm run lint

Only build, for Mac OS: 
>$ npm run build

for windows:
>$ npm run build-cmd

### CDN
Taobao cdn:
>npm config set registry https://registry.npm.taobao.org 

origin:
>npm config set registry http://registry.npmjs.org
